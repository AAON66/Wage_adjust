**1. backend/app/services/feishu_service.py** — 飞书 API 集成服务

```python
class FeishuService:
    FEISHU_BASE_URL = 'https://open.feishu.cn/open-apis'
    TOKEN_REFRESH_BUFFER = 300  # 提前 5 分钟刷新
    MAX_RETRIES = 3
    RETRY_DELAYS = [5, 15, 45]  # 递增间隔秒

    def __init__(self, db: Session):
        self.db = db
        self._token: str | None = None
        self._token_expires_at: float = 0
```

核心方法:

**_ensure_token(app_id, app_secret) -> str** (per D-02)
- POST `{FEISHU_BASE_URL}/auth/v3/tenant_access_token/internal`
- 缓存 token 和过期时间，提前 TOKEN_REFRESH_BUFFER 秒刷新
- 检查响应 `code == 0`，否则 raise RuntimeError 含 msg
- 使用 `httpx.post(url, json=..., timeout=10)`

**_fetch_all_records(token, app_token, table_id, field_mapping, since=None) -> list[dict]** (per ATT-01)
- POST `{FEISHU_BASE_URL}/bitable/v1/apps/{app_token}/tables/{table_id}/records/search`
- **【Review #8 fix】完整分页处理**: page_size=500, 循环 page_token 直到 has_more=False
- automatic_fields=True 以获取 last_modified_time
- 若 since 非 None，添加 filter: `field_name='最后修改时间', operator='isGreater', value=[str(since)]`
- 若 filter 不生效（code!=0 且含 field 相关错误），**降级为全量拉取 + 应用层过滤**（RESEARCH 开放问题 #1）
- 返回 list[dict]，每条包含 mapped 后的字段 + feishu_record_id + last_modified_time

**_map_fields(raw_fields: dict, field_mapping: dict) -> dict | None**
- 根据 field_mapping JSON（格式 `{"飞书字段名": "系统字段名"}`）映射字段
- 必须找到 employee_no 映射目标（per D-04），否则返回 None
- **类型强制转换**: attendance_rate/absence_days/overtime_hours -> float, late_count/early_leave_count -> int
- 单条记录转换失败返回 None（不中断整体同步）

**sync_attendance(mode: str, triggered_by: str | None = None) -> FeishuSyncLog** (per D-07, D-08)
- 创建 FeishuSyncLog(status='running')，flush 获取 id
- 获取 FeishuConfig（is_active=True 的配置，仅一条）
- 解密 app_secret（使用 FEISHU_ENCRYPTION_KEY + encrypt_national_id 函数族）
- 获取 token
- 若 mode='incremental'：查询上次成功同步的 finished_at，转为毫秒时间戳作为 since；**增量使用 overlap window（上次时间 - 5 分钟）**以防边界遗漏
- 若 mode='full'：since=None
- 调用 _fetch_all_records
- **【Review #3 fix】事务边界**: upsert 操作在单个事务内完成：
  1. 查询所有 employee_no -> employee_id 映射（一次性 SELECT 全表 Employee.employee_no, Employee.id）
  2. 对每条记录：匹配 employee_no，匹配到则 upsert（按 employee_id+period 或 feishu_record_id），匹配不到则计入 unmatched
  3. **data_as_of 赋值规则**: 若记录有 source_modified_at（last_modified_time），则 `data_as_of = datetime.fromtimestamp(source_modified_at / 1000, tz=timezone.utc)`；否则 `data_as_of = synced_at`（即当前时间）
  4. 所有 upsert 完成后，更新 FeishuSyncLog 为 success，设置 finished_at 和各计数字段
  5. **单次 db.commit()** 提交全部记录 + 日志状态
- 若异常：db.rollback()，用新 session 或 flush+commit 仅保存失败状态的日志
- **【Review #9 fix】未匹配处理**: unmatched_count 和 unmatched_employee_nos（JSON list，最多记录前 100 个未匹配工号）存入 FeishuSyncLog

**sync_with_retry(mode, triggered_by) -> FeishuSyncLog** (per D-09, ATT-07)
- 重试 3 次，间隔 [5, 15, 45] 秒
- 每次重试 time.sleep(delay)
- 全部失败后记录最终错误到 FeishuSyncLog
- 返回最终的 FeishuSyncLog

**create_config(data: FeishuConfigCreate) -> FeishuConfig**

**2. backend/app/scheduler/feishu_scheduler.py** — APScheduler 定时同步（per D-05, D-06, ATT-03）

```python
from __future__ import annotations
import logging
from apscheduler.schedulers.asyncio import AsyncIOScheduler
from apscheduler.triggers.cron import CronTrigger

logger = logging.getLogger(__name__)
scheduler = AsyncIOScheduler()

def start_scheduler(sync_hour: int = 6, sync_minute: int = 0, sync_timezone: str = 'Asia/Shanghai') -> None:
    """启动定时同步调度器。【Review #2 fix】显式使用配置的时区。"""
    scheduler.add_job(
        run_incremental_sync,
        trigger=CronTrigger(hour=sync_hour, minute=sync_minute, timezone=sync_timezone),
        id='feishu_attendance_sync',
        replace_existing=True,
    )
    scheduler.start()
    logger.info('Feishu scheduler started: %02d:%02d %s', sync_hour, sync_minute, sync_timezone)

def stop_scheduler() -> None:
    if scheduler.running:
        scheduler.shutdown(wait=False)

def reload_scheduler(sync_hour: int, sync_minute: int, sync_timezone: str) -> None:
    """配置更新后重新加载调度器。"""
    if scheduler.running:
        scheduler.remove_job('feishu_attendance_sync')
    scheduler.add_job(
        run_incremental_sync,
        trigger=CronTrigger(hour=sync_hour, minute=sync_minute, timezone=sync_timezone),
        id='feishu_attendance_sync',
        replace_existing=True,
    )
    if not scheduler.running:
        scheduler.start()

def run_incremental_sync() -> None:
    """定时增量同步 job。在独立 Session 中执行（per 陷阱 1）。"""
    from backend.app.core.database import SessionLocal
    db = SessionLocal()
    try:
        from backend.app.services.feishu_service import FeishuService
        service = FeishuService(db)
        service.sync_with_retry(mode='incremental', triggered_by='scheduler')
    except Exception:
        logger.exception('定时考勤同步失败')
    finally:
        db.close()
```

        if _config:
            start_scheduler(_config.sync_hour, _config.sync_minute, _config.sync_timezone)
        else:
            logger.info('No active Feishu config found, scheduler not started')
    finally:
        _sched_db.close()
except Exception:
    logger.warning('Failed to start Feishu scheduler, attendance sync disabled', exc_info=True)
