Backend API endpoints:
```
GET  /api/v1/feishu/config          → FeishuConfigRead (admin, hrbp)
GET  /api/v1/feishu/config-exists   → {exists: bool} (admin, hrbp)
POST /api/v1/feishu/config          → FeishuConfigRead (admin)
PUT  /api/v1/feishu/config/{id}     → FeishuConfigRead (admin)
POST /api/v1/feishu/sync            → {sync_log_id, status, message} (admin, hrbp)
     body: {mode: 'full' | 'incremental'}
GET  /api/v1/feishu/sync-logs       → SyncLogRead[] (admin, hrbp)
GET  /api/v1/feishu/sync-status     → SyncLogRead | null (admin, hrbp)
GET  /api/v1/attendance/             → {items: AttendanceRecordRead[], total: int} (admin, hrbp)
     query: search, department, page, page_size
GET  /api/v1/attendance/{employee_id} → AttendanceSummaryRead | {data: null, message: str} (admin, hrbp, manager)
     query: period (optional, default latest)
```

- **请求取消**: useEffect 返回 abort controller 清理，防止快速切换员工时旧请求覆盖新数据（Review LOW concern）
- 错误不向上冒泡，不影响父组件（per ATT-07）
