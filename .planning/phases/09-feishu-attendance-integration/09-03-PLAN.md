API 端点:
- GET  /api/v1/feishu/config-exists -> FeishuConfigExistsResponse { configured: bool, has_synced: bool }
- GET  /api/v1/feishu/config -> FeishuConfigRead (admin only)
- POST /api/v1/feishu/config -> FeishuConfigRead (body: FeishuConfigCreate, admin only)
- PUT  /api/v1/feishu/config/{id} -> FeishuConfigRead (body: FeishuConfigUpdate, admin only)
- POST /api/v1/feishu/test-connection -> { success: bool, message: str } (body: TestConnectionRequest optional)
- POST /api/v1/feishu/sync -> SyncResultResponse (body: { mode: 'full' | 'incremental' })
- GET  /api/v1/feishu/sync-status -> SyncStatusRead
- GET  /api/v1/attendance/{employee_id} -> AttendanceSummary
- GET  /api/v1/attendance/?department_id=&search=&offset=&limit= -> { items: AttendanceRecordRead[], total: int }

interface FieldMapping {
