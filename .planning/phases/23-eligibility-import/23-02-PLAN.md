| 前端请求 → API router | 需要 JWT 认证 + 角色验证 |

| T-23-07 | Tampering | FeishuSyncRequest.field_mapping | mitigate | Pydantic schema 验证 Dict[str, str] 类型 |
| T-23-08 | Denial of Service | POST /feishu/sync | mitigate | Celery task soft_time_limit=600 + 限流器 |
