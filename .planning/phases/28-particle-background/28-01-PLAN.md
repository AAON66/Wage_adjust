       // 解析 hex -> rgb
       function hexToRgb(hex: string): { r: number; g: number; b: number } {
         const clean = hex.replace('#', '');
         const full = clean.length === 3 ? clean.split('').map((c) => c + c).join('') : clean;
         const r = parseInt(full.substring(0, 2), 16);
         const g = parseInt(full.substring(2, 4), 16);

    5. `function resize()`：
       ```ts
       const dpr = window.devicePixelRatio || 1;
       const cssW = window.innerWidth;
       const cssH = window.innerHeight;
       canvas.width = Math.floor(cssW * dpr);
       canvas.height = Math.floor(cssH * dpr);
       canvas.style.width = cssW + 'px';
       canvas.style.height = cssH + 'px';
       ctx.setTransform(1, 0, 0, 1, 0, 0);
       ctx.scale(dpr, dpr);
       // rebuild particles
       const count = Math.max(MIN_PARTICLES, Math.min(MAX_PARTICLES, Math.floor(cssW * cssH / DENSITY_DIVISOR)));
       particles = Array.from({ length: count }, () => ({
         x: Math.random() * cssW,
         y: Math.random() * cssH,
         vx: (Math.random() - 0.5) * 0.6,
         vy: (Math.random() - 0.5) * 0.6,

       ```
       （per D-05 + D-11）
    6. `function draw()`：
       ```ts
       const cssW = window.innerWidth;
       const cssH = window.innerHeight;
       ctx.clearRect(0, 0, cssW, cssH);

       // update + draw particles
       for (const p of particles) {
         // mouse repulsion (per D-07, D-08)
         const dxm = p.x - mouseX;
         const dym = p.y - mouseY;
         const distM = Math.hypot(dxm, dym);
         if (distM < INTERACTION_RADIUS && distM > 0.001) {
           const force = (INTERACTION_RADIUS - distM) / INTERACTION_RADIUS; // 0..1
           p.x += (dxm / distM) * force * 2.5;
           p.y += (dym / distM) * force * 2.5;
         }

         // linear motion
         p.x += p.vx;
         p.y += p.vy;

         // bounce at edges
         if (p.x < 0 || p.x > cssW) { p.vx *= -1; p.x = Math.max(0, Math.min(cssW, p.x)); }
         if (p.y < 0 || p.y > cssH) { p.vy *= -1; p.y = Math.max(0, Math.min(cssH, p.y)); }

         // draw particle
         ctx.beginPath();
         ctx.arc(p.x, p.y, p.radius, 0, Math.PI * 2);
         ctx.fillStyle = `rgba(${PR},${PG},${PB},${PARTICLE_BASE_ALPHA})`;
         ctx.fill();
       }

       // draw links (per D-06)
       for (let i = 0; i < particles.length; i++) {
         for (let j = i + 1; j < particles.length; j++) {
           const a = particles[i];
           const b = particles[j];
           const dx = a.x - b.x;
           const dy = a.y - b.y;
           const dist = Math.hypot(dx, dy);
           if (dist < LINK_DISTANCE) {
             const alpha = (1 - dist / LINK_DISTANCE) * LINK_BASE_ALPHA;
             ctx.strokeStyle = `rgba(${PR},${PG},${PB},${alpha})`;
             ctx.lineWidth = 1;
             ctx.beginPath();
             ctx.moveTo(a.x, a.y);
             ctx.lineTo(b.x, b.y);
             ctx.stroke();
           }
         }
       }
       ```

## Trust Boundaries

本组件为纯前端装饰层，无用户输入渲染、无数据持久化、无 API 调用。主要信任边界仅为 DOM/浏览器 API 边界。

| Boundary | Description |
|----------|-------------|
| 浏览器 API → 组件 | `matchMedia`、`requestAnimationFrame`、`visibilitychange` 等 API 可用性与行为差异 |
| DOM → Canvas 2D Context | `canvas.getContext('2d')` 可能返回 null（极罕见，如无痕模式限制） |
| 用户 Input Device → mousemove | 高频 mousemove 事件可能带来性能压力 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-28-01 | Denial of Service (performance) | mousemove 事件处理 | accept | mousemove 只写两个 number 到闭包变量，不触发重渲染；实际绘制节奏由 rAF 决定，浏览器已内置到每帧一次。无需额外 debounce。若未来实测高频 mousemove 有掉帧，再引入 rAF-scheduled 的 mouse state sync。 |
| T-28-02 | Denial of Service (memory leak) | rAF / event listeners / matchMedia listener | mitigate | 两个 useEffect 均 return cleanup 函数，显式移除 resize / visibilitychange / mousemove / mouseout / matchMedia change 全部事件；cancelAnimationFrame 在 stop/unmount 双路径调用。验证点：grep `return () => {` 后必须伴随至少 4 处 removeEventListener + 1 处 cancelAnimationFrame。 |
| T-28-03 | Denial of Service (rendering failure) | canvas.getContext('2d') 返回 null | mitigate | 若 ctx === null，useEffect 直接 return 不启动动画；登录页 fallback 为既有 `--color-bg-page` 底色（Canvas 元素虽存在但为空，不遮挡）。用户体验等同 reduced-motion 模式。 |
| T-28-04 | Denial of Service (infinite loop on coarse pointer) | touchmove 误当 mousemove | mitigate | `(pointer: coarse)` 命中时不绑定 mousemove 也不绑定 touchmove，避免移动端滚动被干扰（per D-09）。 |
| T-28-05 | Information Disclosure | 无 | N/A | 组件不读取任何用户数据、不发起任何网络请求；Canvas 内容纯算法生成，不涉及 PII/凭证。 |
| T-28-06 | Tampering | 无 | N/A | 组件不接收外部 props、不持久化状态；视觉可被浏览器 DevTools 修改属于允许行为，无业务影响。 |
| T-28-07 | Spoofing / Elevation of Privilege | 无 | N/A | 组件 `pointer-events: none` + `aria-hidden`，不参与事件分发也不出现在 a11y 树，无权限面。 |
