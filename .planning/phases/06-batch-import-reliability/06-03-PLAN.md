POST /api/v1/imports/jobs?import_type=employees
- 成功: 201 + ImportJobRead JSON
- 部分失败: 207 + ImportJobRead JSON
- 全部失败: 207 + ImportJobRead JSON
- 请求错误: 400 + {detail: string}

ImportJobRead 响应结构:
```json
{

  "success_rows": 90,
  "failed_rows": 10,
  "result_summary": {
    "rows": [
      {"row_index": 1, "status": "success", "message": "导入成功。"},
      {"row_index": 2, "status": "failed", "message": "部门\"xxx\"未创建...", "error_column": "所属部门"}
    ],
    "supported_types": ["certifications", "employees"]
  },
  "created_at": "2026-03-28T..."
}
```

GET /api/v1/imports/templates/{import_type}?format=xlsx|csv
- 返回文件流（xlsx 或 csv）

GET /api/v1/imports/jobs/{job_id}/export?format=xlsx|csv
- 返回错误报告文件流（xlsx 或 csv）

```typescript
function saveBlob(blob: Blob, filename: string) {
  const url = URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = filename;
  document.body.appendChild(a);
  a.click();
  document.body.removeChild(a);
  URL.revokeObjectURL(url);
}
```
