       - `from __future__ import annotations`
       - Import: `dataclass` from dataclasses, `date` from datetime
       - Define grade order: `GRADE_ORDER: dict[str, int] = {'A': 5, 'B': 4, 'C': 3, 'D': 2, 'E': 1}`
       - `@dataclass(frozen=True) class EligibilityThresholds`: min_tenure_months (int, default 6), min_adjustment_interval_months (int, default 6), performance_fail_grades (tuple[str, ...], default ('C','D','E')), max_non_statutory_leave_days (float, default 30.0)
       - `@dataclass(frozen=True) class RuleResult`: rule_code (str), rule_label (str), status (str: 'eligible'|'ineligible'|'data_missing'), detail (str)
       - `@dataclass class EligibilityResult`: overall_status (str: 'eligible'|'ineligible'|'pending'), rules (list[RuleResult])
       - `class EligibilityEngine`:
         - `__init__(self, thresholds: EligibilityThresholds | None = None)` — defaults to EligibilityThresholds()
         - `check_tenure(self, hire_date: date | None, reference_date: date) -> RuleResult` — if None return data_missing; compute month diff = (ref.year - hire.year)*12 + (ref.month - hire.month); >= threshold = eligible else ineligible
         - `check_adjustment_interval(self, last_adjustment_date: date | None, reference_date: date) -> RuleResult` — if None return data_missing; same month diff logic; rule_code='ADJUSTMENT_INTERVAL', rule_label='调薪间隔'
         - `check_performance(self, grade: str | None) -> RuleResult` — if None return data_missing; if grade in performance_fail_grades return ineligible; else eligible; rule_code='PERFORMANCE', rule_label='绩效等级'
         - `check_leave(self, non_statutory_leave_days: float | None) -> RuleResult` — if None return data_missing; if > threshold return ineligible; <= threshold eligible; rule_code='LEAVE', rule_label='非法定假期'
         - `evaluate(self, *, hire_date, last_adjustment_date, performance_grade, non_statutory_leave_days, reference_date) -> EligibilityResult` — calls all 4 check methods, computes overall_status per D-10: any ineligible -> 'ineligible'; all eligible -> 'eligible'; else -> 'pending'
       - NO database imports, NO session dependencies. Pure computation only.
