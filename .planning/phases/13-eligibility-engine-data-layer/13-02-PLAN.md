    1. Create `backend/app/services/eligibility_service.py`:
       - `from __future__ import annotations`
       - `class EligibilityService`:
         - `__init__(self, db: Session, *, settings: Settings | None = None)` — accepts optional settings for testability
         - `_build_thresholds(self) -> EligibilityThresholds` — converts Settings fields to EligibilityThresholds dataclass. Parse `eligibility_performance_fail_grades` string ("C,D,E") into tuple.
         - `check_employee(self, employee_id: str, *, reference_date: date | None = None, year: int | None = None) -> EligibilityResult`:
           - Default reference_date to `date.today()` if None
           - Default year to `reference_date.year - 1` if None (previous year performance)
           - Query Employee by id, raise HTTPException(404) if not found
           - Get hire_date from Employee.hire_date
           - Get last_adjustment_date: query `SalaryAdjustmentRecord` for max(adjustment_date) where employee_id matches. If no record, fall back to Employee.last_salary_adjustment_date (per research recommendation).
           - Get performance_grade: query `PerformanceRecord` where employee_id and year match. Return grade or None.
           - Get non_statutory_leave_days: query `AttendanceRecord` where employee_id matches and period starts with str(year). Sum `non_statutory_leave_days` across periods. If no records or all None, pass None.
           - Instantiate `EligibilityEngine(thresholds=self._build_thresholds())`
           - Call `engine.evaluate(...)` with collected data
           - Return result

         - `create_performance_record(self, *, employee_no: str, year: int, grade: str, source: str = 'manual') -> PerformanceRecord`:
           - Validate grade in ('A','B','C','D','E')
           - Look up Employee by employee_no, raise 404 if not found
           - Upsert: check existing by (employee_id, year). If exists, update grade and source. If not, create new.
           - Flush, return record.

         - `create_salary_adjustment_record(self, *, employee_no: str, adjustment_date: date, adjustment_type: str, amount: Decimal | None = None, source: str = 'manual') -> SalaryAdjustmentRecord`:
           - Validate adjustment_type in ('probation', 'annual', 'special')
           - Look up Employee by employee_no, raise 404 if not found
           - Create new record (no upsert -- multiple adjustments allowed)
           - Flush, return record.

         - `list_performance_records(self, employee_id: str) -> list[PerformanceRecord]` — query by employee_id ordered by year desc
         - `list_salary_adjustment_records(self, employee_id: str) -> list[SalaryAdjustmentRecord]` — query by employee_id ordered by adjustment_date desc
