| Label | 12px | 400 | 1.5 |
| Heading (section) | 15px | 600 | 1.4 |

Weights declared: **2** (400 regular, 600 semibold).

Labels at 12px are visually distinct from 13.5px body text through size alone; a separate 500 weight is unnecessary.

## Interaction Contracts

### Flow 1: Duplicate Detection on File Select (SHARE-01)

| Step | Trigger | UI Response |
|------|---------|-------------|
| 1 | User selects file(s) via file input | Show inline loading indicator ("正在检查文件...") next to file name |
| 2 | Frontend computes SHA-256 via `crypto.subtle.digest` | No visible change (computation is fast for files under 50MB) |
| 3 | Frontend calls `POST /api/v1/files/check-duplicate` with `{ content_hash }` | Spinner continues |
| 4a | Response: `is_duplicate: false` | Proceed with normal upload flow. No modal. |
| 4b | Response: `is_duplicate: true` | Show `DuplicateWarningModal` with original uploader info |

For batch uploads (D-04): process files sequentially. Show modal for each duplicate file one at a time. User resolves each before the next is checked.

### Flow 2: Duplicate Warning Modal (SHARE-01, D-02)

**Layout:** Centered modal overlay (reuse `ModalOverlay` pattern). Max-width 420px.

| Element | Specification |
|---------|---------------|
| Icon | Warning triangle icon, 32px, color `--color-warning` |
| Title | "文件内容重复", 18px, weight 600, color `--color-ink` |
| Body text | "此文件已由 **{uploader_name}** 于 **{uploaded_date}** 提交。继续上传将自动向对方发起共享申请。", 13.5px, weight 400, color `--color-steel`, line-height 1.6 |
| File name | Displayed below body, 12px monospace, color `--color-ink`, word-break: break-all |
| Button row | Right-aligned, 8px gap |
| Cancel button | `action-secondary`, label "取消" |
| Confirm button | `action-primary`, label "继续上传" |
| Backdrop click | Closes modal (same as cancel) |
| Keyboard | Escape closes modal. Enter triggers "继续上传" when focused. |

### Flow 3: Sharing Request Auto-Creation (SHARE-02, D-06)

| Step | Trigger | UI Response |
|------|---------|-------------|
| 1 | User clicks "继续上传" in duplicate modal | Modal closes. File uploads normally. |
| 2 | Upload API returns success with `sharing_request_created: true` | Show toast/inline success: "文件已上传，共享申请已发送给 {uploader_name}" |
| 3 | Toast auto-dismisses after 4 seconds | No further action needed |

Toast style: Bottom-right position, `--color-info-bg` background, `--color-info` text, 13px, 8px 16px padding, border-radius 6px, shadow `--shadow-dropdown`.

### Flow 4: Sharing Requests Page (SHARE-03, D-07, D-08)

**Page layout:** Standard page structure matching existing pages.

| Element | Specification |
|---------|---------------|
| Page eyebrow | "文件共享", `eyebrow` class |
| Page title | "共享申请", `page-title` class |
| Page description | "查看和管理文件共享申请。当他人上传与您相同内容的文件时，系统会自动发起共享申请。", `page-desc` class |
| Tab bar | Two chip-button tabs: "收到的申请" (default active) / "发出的申请" |
| Request list | `table-shell` + `table-lite` table layout |

**Table columns (incoming "收到的申请" tab):**

| Column | Width | Content |
|--------|-------|---------|
| 申请人 | auto | Requester name, 13.5px |
| 文件名 | auto | Original file name, 13.5px, truncate with ellipsis at 200px |
| 申请日期 | 120px | YYYY-MM-DD format, 13px, color `--color-steel` |
| 建议比例 | 100px | "{proposed_pct}% : {100-proposed_pct}%", 13px |
| 状态 | 100px | `status-pill` with status-specific colors (see below) |
| 操作 | 160px | Action buttons (see below) |

**Table columns (outgoing "发出的申请" tab):**

| Column | Width | Content |
|--------|-------|---------|
| 原上传者 | auto | Original uploader name, 13.5px |
| 文件名 | auto | File name, 13.5px, truncate with ellipsis at 200px |
| 申请日期 | 120px | YYYY-MM-DD format |
| 建议比例 | 100px | "{proposed_pct}% : {100-proposed_pct}%" |
| 状态 | 100px | `status-pill` |
| 最终比例 | 100px | Show only if approved, "{final_pct}% : {100-final_pct}%" |

**Status pill styles:**

| Status | Background | Text Color | Label |
|--------|------------|------------|-------|
| pending | `--color-warning-bg` (#FFF3E8) | `--color-warning` (#FF7D00) | 待审批 |

| rejected | `--color-danger-bg` (#FFECE8) | `--color-danger` (#F53F3F) | 已拒绝 |
| expired | `--color-bg-subtle` (#F2F3F5) | `--color-steel` (#646A73) | 已超时 |

### Flow 5: Approve/Reject Actions (SHARE-03, SHARE-04, D-08, D-12)

**Actions column for pending incoming requests:**

| Button | Class | Label | Behavior |
|--------|-------|-------|----------|
| Approve | `action-primary` (height 28px, font 12.5px) | "审批申请" | Opens inline ratio editor row below the table row |
| Reject | `action-danger` (height 28px, font 12.5px) | "拒绝" | Shows confirmation via `window.confirm("确认拒绝此共享申请？拒绝后对方将无法再次申请。")` |

**Inline ratio editor (shown after clicking "审批申请"):**

| Element | Specification |
|---------|---------------|
| Container | Expands below the row, `surface-subtle` background, 16px padding, 8px border-radius |
| Label | "调整贡献比例", 12px, weight 400, color `--color-steel` |
| Ratio display | "申请者 {pct}% : 您 {100-pct}%", 13.5px, weight 400, color `--color-ink` |
| Input | `toolbar-input`, type="number", min=1, max=99, step=1, width 80px. Pre-filled with `proposed_pct` (default 50). |
| Confirm button | `action-primary`, height 28px, label "确认审批" |
| Cancel link | Plain text button, 12.5px, color `--color-steel`, label "取消", underline on hover |

For non-pending requests: no action buttons shown. Status pill only.

### Flow 7: Error States

**Hash computation failure:**

| Element | Specification |
|---------|---------------|
| Display | Inline error banner below file input area |
| Style | `rounded-[10px]`, `--color-danger-bg` background, `--color-danger` text, 13px |
| Copy | "文件读取失败，请重新选择文件。如果问题持续，请尝试使用较小的文件。" |

**Check-duplicate API failure:**

| Element | Specification |
|---------|---------------|
| Display | Same inline error banner style |
| Copy | "无法检查文件重复状态，请稍后重试。您也可以直接上传。" |
| Fallback | Proceed with upload without duplicate check (degrade gracefully) |

**Sharing request API failure (approve/reject):**

| Element | Specification |
|---------|---------------|
| Display | Inline error text below the action buttons |
| Style | 12px, color `--color-danger` |
| Copy | "操作失败，请稍后重试。" |
