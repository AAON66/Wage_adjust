<!-- Plan 02 冻结的 API contract (backend) -->

GET /api/v1/performance/records/by-employee/{employee_id}
→ PerformanceHistoryResponse { items: PerformanceRecordItem[] }
  PerformanceRecordItem 在本 plan 后：
    id, employee_id, employee_no, employee_name,
    year (number),
    grade (string),
    source (string),
    department_snapshot (string | null),
    comment (string | null),     ← Plan 01 新增字段
    created_at (ISO string)

响应状态：
- 200 → 含 items[]（空数组也 200）
- 401 → 未登录（由 api.ts axios interceptor 处理 refresh）
- 403 → manager 跨部门 或 employee 角色
- 404 → employee_id 不存在
