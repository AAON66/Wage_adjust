## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client -> generate/regenerate | 用户触发 AI 评估异步任务 |
| client -> create_import_job | 用户上传文件触发异步导入 |
| API -> Redis broker | 文件 base64 内容在 Redis 中短暂存储 |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-22-05 | Spoofing | generate_evaluation | mitigate | 保留 AccessScopeService.ensure_submission_access 同步校验，仅授权用户可触发 task |
| T-22-06 | Spoofing | create_import_job | mitigate | 保留 require_roles('admin', 'hrbp', 'manager') 角色校验 |

| T-22-08 | Information Disclosure | file_bytes_b64 in Redis | accept | 导入文件（员工/认证数据）在 Redis 中暂存直到 task 消费完毕，Redis 仅内网访问 |
