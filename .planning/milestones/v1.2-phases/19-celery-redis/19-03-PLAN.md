## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| parent process -> prefork worker child | SQLAlchemy 连接池对象跨进程继承，若 bind 错配会留下失效连接 |
| backend probe -> Redis broker -> celery worker | 任务消息跨进程/跨容器流转，需要真实消费证据而不是本地 `.apply()` 假阳性 |
| verification artifact -> requirements traceability | 只有真实运行时证据落地后，需求状态才允许从 Pending 变为 Complete |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
