### FeishuFieldMapper Layout

```
+----------------------------------------------------------+
| position: relative; (container ref for SVG overlay)       |
| ┌─ 左栏 (48%) ─────┐  gap  ┌─ 右栏 (48%) ─────┐        |
| │ .section-note:    │       │ .section-note:    │        |
| │ "飞书字段"         │       │ "系统字段"         │        |
| │                   │       │                   │        |
| │ ┌─ field item ──┐│       │┌─ field item ──┐  │        |
| │ │ field_name    ││       ││ employee_no   │  │        |
| │ │ (draggable)   ││  SVG  ││ (drop target) │  │        |
| │ └───────────────┘│ lines │└───────────────┘  │        |
| │ ┌─ field item ──┐│       │┌─ field item ──┐  │        |
| │ │ field_name_2  ││ ───── ││ year          │  │        |
| │ │ (draggable)   ││       ││ (drop target) │  │        |
| │ └───────────────┘│       │└───────────────┘  │        |
| │ ...              │       │ ...              │        |
| └──────────────────┘       └──────────────────┘        |
|                                                          |
| SVG overlay: absolute, inset-0, pointer-events-none      |
| <line> elements: stroke=var(--color-primary), width=2    |
+----------------------------------------------------------+
| 已建立 {N} 个映射关系        [清除全部映射] .chip-button    |
+----------------------------------------------------------+
```

**Drag interaction contract:**
- Left column items: `draggable="true"`, set `dataTransfer` with field ID on `dragstart`
- Right column items: `onDragOver` (preventDefault), `onDrop` (read field ID, create connection)
- Connected items: show `--color-primary-light` background, `--color-primary-border` border
- Unconnected items: default `.surface-subtle` background
- Remove connection: click the connected line or click a "x" icon on the connected field item
- SVG line coordinates: calculated via `getBoundingClientRect()` relative to container ref
- Recalculate on window resize via `ResizeObserver`

Source: D-06 (拖拽连线设计), RESEARCH.md Pattern 4 / Code Example.

## Interaction States

### Tab Navigation

| State | Visual |
|-------|--------|
| Default | `color: var(--color-steel)`, `border-bottom: 2px solid transparent` |
| Active | `color: var(--color-primary)`, `border-bottom: 2px solid var(--color-primary)` |
| Hover (inactive) | `color: var(--color-ink)` |

Source: Existing EligibilityManagementPage tab pattern.

### Excel Import Flow

```
[Idle] --> [File Selected] --> [Importing...] --> [Result]
  ^                                                  |
  +------ [Reset / Import Another] -----------------+
```

| State | Button State | Drop Zone State |
|-------|-------------|-----------------|
| Idle | "开始导入" disabled | Empty, awaiting file |
| File Selected | "开始导入" enabled | Shows filename + size |
| Importing | "开始导入" disabled + spinner | Hidden or disabled |
| Result | "开始导入" re-enabled | Reset to empty |

### Feishu Sync Flow

```
[Enter URL] --> [获取字段] --> [Map Fields] --> [开始同步] --> [Syncing...] --> [Result]
```

| State | UI |
|-------|----|
| No URL | URL input empty, "获取字段" disabled |
| URL entered | "获取字段" enabled |
| Fetching fields | "获取字段" disabled + spinner, loading skeleton in mapper area |
| Fields loaded | FeishuFieldMapper visible with left/right columns |
| Mapping incomplete | "开始同步" disabled, status: "已建立 0 个映射关系" |
| Mapping complete | "开始同步" enabled, status: "已建立 N 个映射关系" |
| Syncing | "开始同步" disabled + spinner, progress text below |
| Result | ImportResultPanel shown |

### Drag States (FeishuFieldMapper)

| State | Left Item (Feishu) | Right Item (System) | Line |
|-------|--------------------|---------------------|------|
| Default | `.surface-subtle`, cursor: grab | `.surface-subtle` | none |
| Dragging | opacity: 0.5 | -- | none |
| Drag over valid target | -- | border: `var(--color-primary)`, bg: `var(--color-primary-light)` | none |
| Connected | bg: `var(--color-primary-light)`, border: `var(--color-primary-border)` | bg: `var(--color-primary-light)`, border: `var(--color-primary-border)` | SVG `<line>` stroke: `var(--color-primary)`, strokeWidth: 2 |
| Hover connected item | show "x" remove icon | show "x" remove icon | stroke opacity: 0.6 |

## Loading & Progress

| Scenario | Pattern |
|----------|---------|
| Fetching feishu fields | Skeleton placeholder (3 rows, 60% width gray bars) in mapper area |
| Excel import in progress | Disable button, show inline spinner + "正在导入数据..." text |
| Feishu sync in progress | Disable button, show inline spinner + progress text from useTaskPolling |
| Import/sync result loading | Fade-in animation (`.animate-fade-soft`) on ImportResultPanel |

Spinner: 16x16 inline SVG, `animate-spin`, stroke: `currentColor`.

## Accessibility

| Requirement | Implementation |
|-------------|----------------|
| Tab keyboard navigation | Tab key cycles through tab buttons, Enter/Space activates |
| Drag-and-drop keyboard fallback | Each left-column item has a "连接到..." button that opens a select dropdown of available right-column fields |
| File drop zone | `role="button"`, `tabIndex={0}`, Enter/Space triggers file picker |
| Progress announcement | `role="status"`, `aria-live="polite"` on progress text |
| Error alerts | `role="alert"` on error messages (already used in ImportResultPanel) |
| Connected field pairs | `aria-label="{feishu_field} 已映射到 {system_field}"` on connected items |
